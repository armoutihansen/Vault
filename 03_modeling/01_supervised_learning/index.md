# 01 — Supervised Learning

Models that learn a mapping from inputs to targets from labelled training data.

## Sublayers

### [[03_modeling/01_supervised_learning/01_linear_and_glm/index|01 — Linear Models and GLMs]]
Ordinary least squares, Ridge, Lasso, logistic regression, and generalised linear models.

### [[03_modeling/01_supervised_learning/02_tree_based_models/index|02 — Tree-Based Models]]
Decision trees, random forests, gradient boosting (XGBoost, LightGBM).

### [[03_modeling/01_supervised_learning/03_kernel_methods/index|03 — Kernel Methods]]
Support vector machines, kernel trick, SVR, kernel ridge regression.

### [[03_modeling/01_supervised_learning/04_instance_based_methods/index|04 — Instance-Based Methods]]
K-nearest neighbours, locally weighted regression.
