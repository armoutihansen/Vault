# Languages

Programming language reference notes — syntax, idioms, and key patterns.
## Go

Compiled, statically typed, concurrency-first language. Well-suited for backend services, CLIs, and data pipelines.

- [[go/go_basics]] — variables, types, fmt formatting
- [[go/go_functions]] — signatures, multiple returns, closures, defer, variadic
- [[go/go_control_flow]] — if/switch/for, range
- [[go/go_structs]] — struct definition, methods, embedding, memory layout
- [[go/go_interfaces]] — implicit implementation, type assertions, type switches
- [[go/go_pointers]] — pointer syntax, pass-by-reference, nil, pointer receivers
- [[go/go_collections]] — slices, maps, arrays
- [[go/go_errors]] — error interface, custom errors, iota
- [[go/go_concurrency]] — goroutines, channels, mutexes
- [[go/go_packages]] — package system, modules, go toolchain
- [[go/go_generics]] — type parameters, constraints
## JavaScript

Single-threaded, event-driven, runs in browser and Node.js. Dynamically typed.

- [[javascript/js_core_language]] — types, variables, coercions, null/undefined
- [[javascript/js_functions]] — arrow functions, closures, scope, `this`
- [[javascript/js_objects_and_classes]] — objects, prototypes, classes, inheritance
- [[javascript/js_collections]] — arrays, maps, sets, loops
- [[javascript/js_async]] — event loop, promises, async/await
- [[javascript/js_modules]] — CommonJS, ESM, bundlers
- [[javascript/js_runtimes]] — Node.js, npm, Deno, Bun

- [[typescript/ts_unions_and_intersections]] — union types, intersections, literal types
- [[typescript/ts_objects_and_interfaces]] — object types, interfaces, discriminated unions
- [[typescript/ts_classes]] — access modifiers, abstract classes, implements
- [[typescript/ts_generics]] — generic functions, constraints, type inference
- [[typescript/ts_type_narrowing]] — narrowing, type predicates, assertions
- [[typescript/ts_advanced_types]] — conditional types, mapped types, infer
- [[typescript/ts_utility_types]] — Partial, Required, Pick, Omit, Record, Readonly
- [[typescript/ts_enums_and_tuples]] — enums, const enums, tuples
- [[typescript/ts_local_dev]] — tsconfig, declaration files, Vite setup
